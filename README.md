# 🚗 Toapanta Hnos | Taller Mecánico Profesional

Landing page profesional para taller mecánico en Quito, Ecuador. Diseño moderno, completamente responsive, con enfoque en conversión y experiencia de usuario.

## 📋 Descripción

Página web desarrollada para **Tecnología Automotriz Toapanta Hnos**, un taller mecánico con más de 15 años de experiencia en el sector automotriz. El sitio está diseñado para generar leads de calidad a través de un formulario de contacto atractivo y un llamado a la acción destacado: **Diagnóstico GRATIS**.

## ✨ Características Principales

### 🎨 Diseño y Estilo
- **Tema oscuro profesional** con acentos en naranja (#E85D04)
- **Gradientes personalizados** en botones, tarjetas y fondos
- **Efectos hover** suaves en todos los elementos interactivos
- **Animaciones de entrada** al hacer scroll (fade-up)
- **Imagen de fondo en Hero** con superposición de degradado
- **Tipografía moderna** (Inter, Google Fonts)

### 📱 Responsive Design
- **Mobile First**: optimizado para todos los dispositivos
- **Menú flotante inferior** en móviles con íconos y texto
- **Menú lateral flotante** en desktop con tooltips
- **Carruseles adaptativos** que cambian cantidad de slides según el ancho
- **Espaciado mejorado** en pantallas grandes para carruseles

### 🧭 Navegación
- **Header simplificado**: solo logo y botón de diagnóstico
- **Menú lateral flotante** en desktop (acceso rápido a secciones)
- **Bottom navigation** en móviles con 5 accesos principales
- **Botón flotante de WhatsApp** con animación pulsante
- **Scroll suave** entre secciones

### 🔄 Carruseles (Swiper.js)
- **Carrusel de marcas** automático e infinito
- **Carrusel de testimonios** automático sin puntos de paginación
- **Espaciado optimizado** en pantallas grandes (más separación entre slides)
- **Sin puntos de paginación** para una experiencia más limpia

### 📊 Secciones del Sitio

1. **Hero** - Imagen de fondo con degradado, título principal, estadísticas y CTA
2. **Servicios** - Tarjetas con diagnóstico computarizado, mecánica general y hojalatería/pintura
3. **Marcas** - Carrusel con las marcas de vehículos que reparan
4. **Cobertura de años** - Rango 1945-2025 con badges representativos
5. **Testimonios** - Carrusel con reseñas de clientes satisfechos
6. **Contacto** - Formulario de captación de leads con envío por correo
7. **CTA Final** - Botón de WhatsApp para acción inmediata

### 📝 Formulario de Contacto
- Campos: nombre, teléfono, email, marca/modelo, mensaje
- Validación de campos obligatorios
- Envío mediante `mailto:` con cuerpo predefinido
- Feedback visual al usuario
- Diseño coherente con la identidad visual

### 🎯 SEO y Metaetiquetas
- Meta description optimizada
- Título descriptivo con palabras clave
- Estructura semántica HTML5
- Atributos `aria-label` para accesibilidad

## 🛠️ Tecnologías Utilizadas

| Tecnología | Uso |
|------------|-----|
| **HTML5** | Estructura semántica del sitio |
| **CSS3** | Estilos, animaciones, gradientes, responsive design |
| **JavaScript** | Interactividad, carruseles, formulario, animaciones |
| **Swiper.js** | Carruseles táctiles y responsivos |
| **Font Awesome 6** | Iconografía profesional |
| **Google Fonts** | Fuente Inter para mejor legibilidad |
